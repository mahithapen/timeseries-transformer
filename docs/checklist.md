# Project Checklist (CS 4782)

## Poster
- Due: April 30 or May 5 (see course schedule for your group)
- Format: PDF, landscape, max 36" x 24"
- Sections: Title/Authors, Intro/Motivation, Methodology, Results, Conclusion, (Optional) Future Work, References
- Design: 24pt+ body text, strong visuals, captions

## 2-Page Report
- Due: May 12
- Max 2 pages (excluding images/references), 11-12pt, single-spaced
- Required sections: Introduction, Chosen Result, Methodology, Results & Analysis, Reflections, References

## GitHub Deliverable
- Due: May 12
- Required structure: `README.md`, `code/`, `data/`, `results/`, `poster/`, `report/`, `LICENSE`, `.gitignore`
- README sections: Introduction, Chosen Result, GitHub Contents, Re-implementation Details, Reproduction Steps, Results/Insights, Conclusion, References, Acknowledgements
